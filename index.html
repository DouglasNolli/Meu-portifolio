Sitema Para Transportadora

Desenvolvi um sistema de banco de dados para uma empresa transportadora , com foco no controle de fretes, motoristas e veículos. Tecnologias utilizadas: HTML, CSS e JavaScript. Atuei na criação das consultas SQL e desenvolvimento da interface web.
Com esse projeto, aprendi na prática como organizar dados, criar relacionamentos entre tabelas e montar páginas responsivas.
